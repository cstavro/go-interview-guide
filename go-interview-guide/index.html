How to Use This Guide

  • Each question has a main prompt — try to solve it first before revealing anything else.
  • Hints are available if you get stuck. Use them sparingly.
  • Follow-ups simulate the interviewer digging deeper. Only read them after your initial answer.
  • Context provides background on why this question matters in real engineering.
  • Code blocks have a copy button — paste them into your IDE and run them.
  • Or click Generate to create a ready-to-go workspace directory with the boilerplate, tests, and a README.
